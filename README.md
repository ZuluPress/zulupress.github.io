# ZuluPress GitHub Pages

Official ZuluPress LLC GitHub Pages site for app information, terms, privacy policy, and developer integrations.

## Files

- `index.html`: landing page
- `terms.html`: Terms of Service
- `privacy.html`: Privacy Policy
- `styles.css`: site styling
- `assets/zulupress-logo.png`: logo asset
- `.nojekyll`: disables Jekyll processing for simple static hosting

## GitHub Pages settings

Repository: `ZuluPress/zulupress.github.io`

Settings -> Pages:

- Source: Deploy from a branch
- Branch: main
- Folder: /root

Expected URLs:

- `https://zulupress.github.io/`
- `https://zulupress.github.io/terms.html`
- `https://zulupress.github.io/privacy.html`
