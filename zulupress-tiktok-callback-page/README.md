# ZuluPress TikTok OAuth Callback Page

Upload the `auth/` folder to the root of `ZuluPress/zulupress.github.io`.

This creates:

`https://zulupress.github.io/auth/tiktok/callback/`

Use that exact URL as the TikTok Login Kit Web redirect URI.

Do not publish secrets, access tokens, refresh tokens, client secrets, or screenshots containing authorization codes.
